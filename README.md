# ProjectEuler

### Notes

- This public project will display up to the first 100 solutions for Project Euler as allowed by the project's guidelines. Any further problems attempted will not have their solutions made public, and will be kept in a separate private repository.

- The code is executed client side, so runtimes measured will be dependent on the user's system specifications.

- There is no server side code, and all plugins are provided as self contasined js files. The code should run locally with no build required.
